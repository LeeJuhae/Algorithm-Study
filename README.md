# :books: 알고리즘 스터디 (2팀)
이 스터디는 지속적인 알고리즘 문제 풀이와 코드 리뷰를 통한 코딩 능력 향상을 목표로 한다.

### :bulb: 진행 방식
1. 주차별 이슈에 각자 정한 문제를 하나씩 댓글로 올린다. 각자 맡은 요일 **하루 전날**까지 올려야한다.

|요일|월요일|화요일|수요일|목요일|금요일|
|---|:---:|:---:|:---:|:---:|:---:|
|담당자|[노경진](https://github.com/kyoung-jnn)|[박제구](https://github.com/Zigje9)|[이도현](https://github.com/ksmfou98)|[정세희](https://github.com/jsh-me)|[추지온](https://github.com/jionchu)|

2. 각자 fork한 레포에서 문제를 풀이한 후 메인 레포에 PR을 올린다.
3. 매주 주말 코드리뷰를 진행하고 완료되면 merge한다.

### :mag: 문제 선택 기준
- [프로그래머스](https://programmers.co.kr/learn/challenges)
- 난이도 `Level 2` 혹은 `Level 3`에서 고르도록 한다.
